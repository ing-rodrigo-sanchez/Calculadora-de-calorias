# 🧮 Calculadora de Calorías | TMB
> **Herramienta web profesional para el cálculo de la Tasa de Metabolismo Basal y gasto energético diario.**

[**🌐 Explora la aplicación en vivo**](https://calculadora.ing-sanchez-vargas.cl/)

---

## 🚀 Sobre el Proyecto
Esta aplicación permite a los usuarios calcular su Tasa de Metabolismo Basal (TMB) utilizando fórmulas estandarizadas, proporcionando una estimación precisa de las calorías necesarias según el nivel de actividad física. Es un proyecto desarrollado con un enfoque en la utilidad práctica y el diseño limpio.

### ✨ Características principales
* **Interfaz Moderna:** Diseño con una paleta de colores optimizada para la legibilidad.
* **Totalmente Responsivo:** Adaptado para una experiencia fluida tanto en computadoras como en dispositivos móviles.
* **Cálculos Dinámicos:** Resultados instantáneos basados en parámetros de edad, peso, altura y sexo.
* **Monetización Activa:** Implementación técnica de unidades de anuncios para la gestión de ingresos.

---

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5 y CSS3 (uso de Flexbox y variables de CSS).
* **Lógica:** JavaScript Vanilla (sin librerías externas para máxima velocidad).
* **Despliegue:** GitHub Pages con configuración de dominio personalizado.
* **Monetización:** Red de anuncios Adsterra.
