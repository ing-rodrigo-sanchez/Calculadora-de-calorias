## 🚀 Sobre el Proyecto
Esta aplicación permite a los usuarios calcular su Tasa de Metabolismo Basal (TMB) utilizando fórmulas estandarizadas, proporcionando una estimación precisa de las calorías necesarias según el nivel de actividad física.

### ✨ Características principales
* **Interfaz Moderna:** Diseño oscuro con acentos verdes para una lectura descansada.
* **Totalmente Responsivo:** Adaptado para una experiencia fluida en smartphones y tablets.
* **Cálculos Dinámicos:** Resultados instantáneos basados en edad, peso, altura y sexo.
* **Monetización Activa:** Integración de anuncios mediante *Social Bar*.

---

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox).
* **Lógica:** JavaScript (Vanilla JS).
* **Despliegue:** GitHub Pages con dominio personalizado.
* **Monetización:** Adsterra Network.
